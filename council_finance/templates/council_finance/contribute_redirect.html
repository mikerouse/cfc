Data Contribution Has Changed

We've replaced the contribute queue with a more efficient system

What's Changed?

The old contribute queue system had several issues including outdated data, lack of real-time updates, and conflicts when multiple users edited the same data. We've replaced it with a better approach.

New System: Edit Directly + Flag Issues

You can now edit council data directly through council pages, and flag any data issues you notice.

How to Contribute Data Now

1

Direct Editing

Visit any council page and click "Edit Council Data" to add or update financial figures and council information directly.

Browse Councils
2

Flag Data Issues

Found incorrect or suspicious data? Use the new flag system to report issues directly to our moderation team for review.

Benefits of the New System

  • Real-Time Updates: Changes appear immediately without waiting in a queue
  • Conflict Resolution: No more issues with multiple people editing the same data
  • Better Quality Control: Flag system allows targeted reporting of data issues
  • Universal Availability: Flagging works across all content types and pages
